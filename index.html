In order to use our tool properly please insert the number of channels for each band with the following manual:

Total number of channels for each band: E - 101, S1 - 108, S2 - 104, C - 143, L - 163

The fields "NchE, NchS1, NchS2, NchC, NchL" should be filled with the desired number of channels you want to simulate

Then the fields "PchE, PchS1, PchS2, PchC, PchL" should be filled with the desired power per channel for each band

Finally, you chose the number of Spans in the field "Ns" and the length of each span in the field "L".
